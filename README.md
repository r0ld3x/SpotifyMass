# Cara penggunaan

Termux - Android (recommended version 0.117)

1. pkg upgrade && update
2. pkg install git
3. pkg install php
4. termux-setup-storage
5. git clone https://github.com/tahaluindo/SpotifyMass
6. cd SpotifyMass
7. php index.php
